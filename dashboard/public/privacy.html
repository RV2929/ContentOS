Privacy Policy

Last updated: July 5, 2026

ContentOS is a personal tool. It does not sell, share, or monetise any data. All processing happens on your own machine.

This Privacy Policy describes how ContentOS handles information when you use the service.

1. Data We Collect

ContentOS collects minimal data necessary to operate the service:

  • YouTube OAuth tokens — stored locally on your machine to authenticate uploads. Never transmitted to third parties outside of Google's APIs.
  • Clip metadata — titles, descriptions, and scheduling information stored locally and synced to a private or public GitHub repository you control.
  • Video files — processed and stored locally on your machine. Not uploaded to any ContentOS server.

2. Data We Do Not Collect

ContentOS does not collect or store:

  • Personal identifying information (name, email, address)
  • Payment information
  • Usage analytics or telemetry
  • Browser cookies or tracking data
  • IP addresses or device identifiers

3. Third-Party Services

ContentOS interacts with the following third-party services:

  • YouTube Data API (Google) — used to upload videos on your behalf. Governed by Google's Privacy Policy.
  • Anthropic Claude API — used to generate clip titles and descriptions. Only clip transcripts are sent, no personal data. Governed by Anthropic's Privacy Policy.
  • GitHub — clip status data may be synced to a GitHub repository you own and control.
  • Vercel — this dashboard is hosted on Vercel. Vercel may collect standard server logs. Governed by Vercel's Privacy Policy.

4. Data Storage

All ContentOS data (video files, tokens, clip metadata) is stored locally on your own machine. The public dashboard on Vercel reads clip status from a GitHub repository you control — no data passes through any ContentOS-operated server.

5. Data Retention

ContentOS does not retain any data on external servers. You can delete all data at any time by removing files from your local machine and your GitHub repository.

6. Security

API tokens and credentials are stored in local files on your machine. You are responsible for securing your local environment. ContentOS does not transmit credentials to any server other than the authorised third-party APIs they belong to.

7. Changes to This Policy

This privacy policy may be updated from time to time. The date at the top of this page reflects the most recent revision.

8. Contact

For any questions about this privacy policy, contact us at the email associated with the ContentOS project.
